LaVella SNS 콘텐츠 전략
📸 인스타그램 피드 전략
  • 주 2-3회 포트폴리오 사진 게시
  • 피드 색감 통일 — 따뜻한 골드/아이보리 계열
  • 문구는 1-2문장 이내로 절제
  • 감성 한 줄 + 웨딩홀명 태그 전략
  • 스토리로 일상/촬영 뒷이야기 보완
🎬 릴스 전략
  • 월 2-4회 — 양보다 질
  • 실제 본식 영상 편집 (15-30초)
  • BGM 선택이 브랜드 이미지 결정
  • 웨딩홀명 검색 유입 노려서 장소 표시
  • 첫 3초 후킹 — 신부 입장 or 감동 순간
📝 블로그 전략
  • 웨딩홀명 + 본식 사진 키워드 SEO 공략
  • "[웨딩홀명] 본식 사진 후기" 제목 패턴
  • 월 2-4개 꾸준한 업로드
  • 패키지 안내 글 1개 상위 노출 관리
  • 고객 동의 후기 활용으로 신뢰도 구축

소통 전략 — 어떤 무드로 쓸까?
✅ LaVella 권장 소통 방식
  • 피드 → 정보 없이 무드만 / 극도로 절제
  • 블로그 → 정보 제공 + 따뜻한 후기 형식
  • DM 답변 → 친절하되 격식 유지
  • 댓글 → 짧게, 진심 있게
  • 스토리 → 가장 자유롭게 / 뒷이야기
💡 "조용한 브랜딩"의 힘
  • 말이 적을수록 사진이 말한다
  • 설명하지 않는 것이 고급스러움
  • 고객이 먼저 DM을 보내게 하는 피드
  • 할인/이벤트 없이도 가치를 유지
  • 일관성이 곧 신뢰 = 예약으로 연결
